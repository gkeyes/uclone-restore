# Alpha 13 Device Test

Focus: launcher icon clarity and glass button colors.

## Launcher icon

1. Install the APK over the previous version.
2. If the launcher keeps the old icon, restart the launcher or reboot once.
3. Verify the UClone icon:
   - foreground cards are not cropped by the launcher mask;
   - edges are crisp at normal launcher size;
   - no white border remains around the icon.

## Buttons

Check App list and backup/data pages:

1. Favorite star selected state should use a subtle amber glass background.
2. Compact secondary buttons such as `还原` should use a white glass capsule with blue text, not a grey-blue filled capsule.
3. Primary destructive and continue actions should keep clear semantic color.
