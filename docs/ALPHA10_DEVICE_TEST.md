# v0.1.0-alpha.10 真机测试说明

目标：验证 D 方案极简图标已替换为 2K 源图，并确认桌面快捷入口继续使用对应目标 App 图标。

## 1. 安装与图标刷新

1. 安装 `v0.1.0-alpha.10` APK 到主系统 user0。
2. 如果桌面仍显示旧 UClone 图标，重启启动器或重启手机后再看；部分启动器会缓存应用图标。
3. 桌面主图标应是深色底，两块竖向圆角矩形，中间一条连接线。

## 2. 桌面长按快捷入口

长按 UClone Restore 图标。

预期：

- 已收藏 App 仍显示快捷入口。
- 每个快捷入口仍使用对应目标 App 的图标。
- 快捷入口文案继续按状态显示「切换 <App 名>」或「还原 <App 名>」。
