# v0.1.0-alpha.7 真机测试说明

目标：验证收藏 App 的桌面长按快捷入口，以及快捷入口与首页「切换/还原」使用同一套状态判断。

## 1. 安装与准备

1. 安装 `v0.1.0-alpha.7` APK 到主系统 user0。
2. 打开 App，授予 KernelSU/Magisk root。
3. 在 App 列表或详情页给 1-4 个常用 App 点星标收藏。
4. 回到桌面。

## 2. 桌面长按入口

长按 UClone Restore 图标。

预期：

- 出现已收藏 App 的快捷入口。
- 未切换的收藏 App 显示为「切换 <App 名>」。
- 已切换的收藏 App 显示为「还原 <App 名>」。
- 如果收藏超过启动器允许数量，只显示前几个收藏 App。

## 3. 一键切换/还原

1. 点击一个「切换 <App 名>」快捷入口。
2. App 应打开到首页，并启动同一个切换任务：先保存当前主系统状态，再恢复分身态。
3. 任务成功后，再次长按桌面图标，该 App 的快捷入口应变为「还原 <App 名>」。
4. 点击「还原 <App 名>」快捷入口，应恢复切换前的主系统状态，并清除还原标记。

## 4. 失败时需要回传

优先回传最新任务日志：

```sh
su -c 'ls -lt /data/adb/uclone/logs | head'
su -c 'cat /data/adb/uclone/logs/<最新日志>.log'
```

同时截图：

- 桌面长按 UClone Restore 图标后的快捷入口。
- 首页「最新任务」卡片。
- 失败提示或任务日志摘要。
