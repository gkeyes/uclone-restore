# Alpha 14 Device Test

Focus: stacked glass-card launcher icon with adaptive-safe framing.

## Launcher icon

1. Install the APK over the previous version.
2. If the launcher keeps the old icon, restart the launcher or reboot once.
3. Verify the UClone icon:
   - icon uses three stacked liquid-glass cards;
   - no letters, numbers, arrows, or black top bars;
   - card edges are not clipped by round or rounded-square launcher masks;
   - small desktop icon remains readable and does not look like a white bordered screenshot.

## Notes

The legacy launcher PNG uses the full icon. The adaptive foreground is intentionally inset more than the legacy icon to avoid clipping on aggressive launcher masks.
