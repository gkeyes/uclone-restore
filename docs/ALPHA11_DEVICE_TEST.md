# v0.1.0-alpha.11 真机测试说明

目标：验证 D 方案图标、目标 App 快捷入口图标，以及切换/还原任务结束后关闭 user10。

## 1. 安装与图标刷新

1. 安装 `v0.1.0-alpha.11` APK 到主系统 user0。
2. 如果桌面仍显示旧 UClone 图标，重启启动器或重启手机后再看。
3. 桌面主图标应是深色底，两块竖向圆角矩形，中间一条连接线。

## 2. 切换/还原后关闭分身

1. 设置页确认「任务完成后关闭分身系统」开启。
2. 首页收藏 App 点击「切换」，或桌面长按快捷入口点击「切换 <App 名>」。
3. 任务完成后检查日志，应出现：

```text
STOP_CLONE_AFTER_TASK=1 reason=switch_restore
STATE_BEFORE_STOP=...
STOP_USER_OUTPUT=...
STATE_AFTER_STOP=...
```

4. 再执行「还原」，任务完成后同样应关闭 user10。

说明：建立/更新主动备份仍只关闭本次由 UClone 后台启动的 user10；切换/还原则按此开关在任务结束后关闭 user10。

## 3. 桌面长按快捷入口

长按 UClone Restore 图标。

预期：

- 已收藏 App 仍显示快捷入口。
- 每个快捷入口仍使用对应目标 App 的图标。
- 快捷入口文案继续按状态显示「切换 <App 名>」或「还原 <App 名>」。
